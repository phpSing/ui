Slider 轮播图UI自动初始化

使用 Usage

全局调用
$(this).mui_slider({obj, callback});
DOM构成(至少)
<div slider>
	<div>
		<a href="图要去的地方" src="图片路径" target="_blank"></a>
		<div caption>图片描述</div>
	</div>
</div>

参数设置 Setting

参数名
值
slider

初始化属性，赋值可选，设置不同样式

a-z 必须
slider-width

slider 视口宽度

int 可选,默认 1200px
slider-height

slider 视口高度

int 可选,默认 300px
slider-items

slider 每组滑动元素数量

int 可选,默认 1
slider-direction

slider 滑动方向

h, v (horizontal,vertical) 可选,默认 h
slider-speed

slider 滑动速度

int 可选,默认 500 ms
slider-duration

slider 滑动间隔

int 可选,默认 3000 ms
slider-nav

slider 控制点,缩略图

dots, thumb, false 可选,默认 dots
slider-navcontrol

slider 前后一张控制

true , false 可选,默认 true
slider-navposLeft

slider 控制箭头离左侧距离

int 可选,默认 0
slider-navposRange

slider 控制箭头之间距离

int 可选,默认 slider宽度

Dialog 对话框UI自动初始化

使用 Usage

全局调用
$(this).mui_dialog({obj, callback});
DOM构成(至少)
<a dialog href="interface.php">对话框链接</a>

参数设置 Setting

参数名
值
dialog

初始化属性，赋值可选，设置不同样式

a-z 必须
dialog-selector

静态数据内容选择器

string (jquery selector) 可选
dialog-data

对话框传递参数

object 可选
dialog-width

对话框宽度

int 可选，默认根据内容
dialog-height

对话框高度

int 可选，默认根据内容
dialog-title

对话框顶部title内容

string 可选，默认不显示
dialog-speed

对话框显示速度

int 可选，默认 500 ms
